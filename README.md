Project Name: BuildYourOwnMeal – Custom Food Ordering App

BuildYourOwnMeal is a web-based application that allows users to customize and order meals in an interactive way. Users can build dishes by selecting ingredients and preferences instead of just browsing items. The goal is to create a real-world food ordering experience with strong frontend logic and clean UI.

API Used
The project uses APIs like DummyJSON or MealDB to fetch food items and categories, including:
* Item name
* Image
* Category
* Price (or simulated)
* Ratings

Features
* Search items with debounce
* Filter by category, veg/non-veg, and price
* Sort by price, name, or popularity
* Meal builder for pizza, pasta, salad, drinks
* Cart system (add, remove, update quantity, total using reduce)
* Loading indicator while fetching data

Technologies Used
* HTML
* CSS 
* JavaScript 
* Fetch API
  
Application Structure
* Home Page: categories, search, filter, sort
* Builder Page: customization logic
* Cart Page: items and total price
